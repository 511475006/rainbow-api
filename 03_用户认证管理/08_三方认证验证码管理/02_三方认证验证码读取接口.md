
> 获取网关设备ID（设备序列号）【Client   Gateway】

获取网关设备ID 
读取网关的设备ID，此处为InPortal的设备序列号。

####URL
<http://$GATEWAY:PORT/api/gateway/id>

####支持格式
`JSON`

####Content-Type
`application/json`

####HTTP请求方式
`GET`

####是否需要登录
`否`

#### 访问授权限制
* 访问级别：[普通接口]
* 频次限制：[否]
* 访问权限：


####请求参数
无

####注意事项
无

####请求消息内容
``` JSON
设备系统配置
```
####返回结果
``` JSON
{ 
    "dev-id":"PL8141311205821"
} 

```
####返回值字段说明
| 返回值字段 | 字段类型 | 字段说明                |
| ---------- | --------:| :---------------------- |
| result     | string   | 如果上报成功，此字段为“ok” </br>如果上报失败，此字段包含错误信息，参见附录一|

####其他
无
