
> 公共配置发布接口

####URL
<http://AUTHSERVER/api/content_sync/public_config>

####支持格式
`JSON`

####Content-Type
`application/json`

####HTTP请求方式
`POST`

####是否需要登录
`是`

#### 访问授权限制
* 访问级别: [普通接口]
* 频次限制: [否]
* 访问权限: ReadPublicConfig


####请求参数

| 请求参数      |    必选 | 类型及范围  | 说明                                |
| ------------- | -------:| :---------- | ----------------------------------- |
| access_token  | true    | string      | OAuth授权后获得。</br>注：通过HTTP认证头携带时省略。 |

####注意事项
无

####请求消息内容
``` JSON
{
"config":"{}"",
"frontEndConfig":"{}"
}
```

####返回结果
``` JSON
{
 "result": "ok"
}

```
####返回值字段说明
无
